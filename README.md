## 📚Project
* The main reason to build this project was to learn DOM manipulation.                    
* This timer uses Pomodoro Technique of working 25 minutes and taking 5 minutes break.

## ⚙️Tech Stack

* **HTML5**
* **CSS3**
* **ES6(Javascript)**


## 🧾Features

- Work time = 25 minutes
- Break time = 5 minutes
- Timer reset button
- Multi platform
